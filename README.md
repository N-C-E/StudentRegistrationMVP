## How to Run

1. Open the project folder in VS Code.  
2. Open the terminal.  
3. Run these commands one by one:

```
python -m venv venv
```
```
venv\Scripts\activate
```
```
pip install mysql-connector-python pwinput
```
```
python app.py
```
